Vertex Protocol Python SDK
==========================

This is the API reference for the `Vertex Protocol API <https://vertex-protocol.gitbook.io/docs/developer-resources/api>`_ Python SDK.

The latest version of the SDK can be found on `PyPI <https://pypi.org/project/vertex-protocol-sdk/>`_, and the source code is on `GitHub <https://github.com/vertex-protocol/python-server-sdk>`_.
